### Setup Web Server

```bash

```

### ALSA

Install ALSA libraries on Ubuntu (or others Linux distributions).

```bash
$ sudo apt -y install libasound2-dev
$ sudo apt -y install libflac-dev
$ sudo apt -y install libfltk1.3-dev
$ sudo apt -y install libsndfile1-dev
```
