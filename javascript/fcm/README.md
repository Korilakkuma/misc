# Web Push by FCM (Firebase Cloud Messaging)

## 1. Install npm modules
  
```bash
$ npm install
```
  
## 2. Add Firebase
  
- [Firebase console](https://console.firebase.google.com/?hl=ja)
  
## 3. Get API key
  
- [Google Developers Console](https://console.developers.google.com/apis/dashboard)
  
## 4. Start server
  
```bash
$ npm start
```
